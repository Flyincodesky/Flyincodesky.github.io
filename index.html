实验室简介

欢迎来到我们的研究实验室！本实验室隶属于某某大学计算机学院，成立于2010年，致力于人工智能、软件工程、分布式系统等前沿领域的研究与创新。我们的使命是通过卓越的科研工作推动技术进步，培养具有国际视野的优秀研究人才。

实验室拥有先进的科研设备和良好的学术氛围，与国内外多所知名高校和研究机构保持密切合作。近年来，我们在顶级国际会议和期刊上发表了多篇高水平学术论文，承担了多项国家级科研项目，取得了一系列重要研究成果。

selected publications

  1. ICSE
    DeepVul: Deep Learning for Vulnerability Detection in Source Code
    Professor Zhang†, Ming Li, and Fang Wang
    In Proceedings of the 48th International Conference on Software Engineering (ICSE), 2026
    Abs Bib PDF Code

    We propose DeepVul, a deep learning-based approach for automatic vulnerability detection in source code. Our method combines static analysis and neural networks to identify security vulnerabilities with high precision and recall. Experimental results on real-world projects demonstrate that DeepVul outperforms state-of-the-art tools by 15-20%.

  2. NeurIPS
    Efficient Neural Network Compression via Adaptive Pruning
    Ming Li†, Professor Zhang, and Wei Chen
    In Advances in Neural Information Processing Systems (NeurIPS), 2025
    NeurIPS 2025 杰出论文提名奖 Abs Bib PDF Code Slides

    Outstanding Paper Nominee

    This paper presents an adaptive pruning method for neural network compression. Our approach dynamically adjusts the pruning rate based on layer importance, achieving 80% model size reduction while maintaining accuracy. Experiments on ImageNet demonstrate superior performance compared to existing methods.

  3. MLSys
    Scalable Distributed Training for Large Language Models
    Professor Zhang†, Fang Wang, and Jing Zhou
    In Proceedings of the 7th Conference on Machine Learning and Systems (MLSys), 2024
    Abs Bib PDF Code
