  1. ICSE
    DeepVul: Deep Learning for Vulnerability Detection in Source Code
    Professor Zhang†, Ming Li, and Fang Wang
    In Proceedings of the 48th International Conference on Software Engineering (ICSE), 2026
    Abs Bib PDF Code

    We propose DeepVul, a deep learning-based approach for automatic vulnerability detection in source code. Our method combines static analysis and neural networks to identify security vulnerabilities with high precision and recall. Experimental results on real-world projects demonstrate that DeepVul outperforms state-of-the-art tools by 15-20%.
